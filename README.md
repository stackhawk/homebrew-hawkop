# homebrew-hawkop

Homebrew tap for [HawkOp](https://github.com/stackhawk/hawkop) — a CLI companion for the StackHawk AppSec Intelligence Platform.

## Install

```sh
brew tap stackhawk/hawkop
brew install hawkop
```

## Upgrade

```sh
brew update
brew upgrade hawkop
```

## Uninstall

```sh
brew uninstall hawkop
brew untap stackhawk/hawkop
```
